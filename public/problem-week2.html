第二周：函数调用 基本语句

by zhengdongjian2013@gmail.com

这周的题目，要求首先能熟练地掌握基本的控制结构语句和函数调用。这里简单地复习一下。

首先是各种循环(while, for, do...while):

for (int i = 0; i < n; ++i) {
    cout << "i = " << i << endl;
}

int n = 12;
while (n > 0) {
    cout << "n = " << n << endl;
}

int n = 12;
do {
    cout << "n = " << n << endl;
} while (n > 0);

需要格外注意的是，开始使用时，可能比较容易将for循环搞错，比如 for (int i = 0; i < 0; i++) {

}

上述语句的执行情况是：①声明i并赋初值0，②判断i<0，为假，循环退出，③i++是在循环内内容执行完第一次后才会做的，此处并不会执行。

然后是函数调用相关:

一般100行以内的小程序，并且涉及的函数不多(不多于5个)的情况下，可以将函数头和函数体写一块，并放在main函数前供调用，否则建议main函数前只写函数头，具体实现放在后面，这样有助于直接看到函数的接口和返回（也就是传入的参数有几个，依次是什么类型，返回值是啥样的这些东西）。代码量继续增大，对程序猿来说，有一个大概1000行的瓶颈，超过这个代码量，你的思维会达到一个极限，会无法理清楚各个模块（说直白一些，就是各个函数）之间的关系，此后就需要做模块分离，将它们各自分离出来，这些在以后的软件工程中会学习得到，就不再废话。。

顺带一提，你有个习惯是在main函数开头声明所有需要用到的变量，这其实是VC6.0才会有的变态限制。比如for循环需要一个局部变量int i，那么其实写成这样：

for (int i = 0; ...; ...;) {

}

比写成：

int n, m, k, v, i;
for (i = 0; ...; ...;) {

}

好得多，代码的层次性也会好一些。
